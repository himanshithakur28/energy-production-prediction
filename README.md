# Energy Production Forecasting

## Overview

This project uses Machine Learning techniques to forecast energy production by analyzing historical energy generation data and identifying patterns that influence future production.

## Problem Statement

The objective of this project is to build a machine learning model that accurately predicts future energy production using historical data and relevant features.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation

## Models Used

- Linear Regression
- Random Forest Regressor
- Decision Tree Regressor
- Other regression algorithms

## Evaluation Metrics

The models were evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Results

Different machine learning models were trained and compared to determine the model that provides the most accurate energy production forecasts.
